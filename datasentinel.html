Production Architecture

DataSentinel runs on an event-driven AWS architecture designed for scalability, reliability, and auditability. 

User Browser

ECS Fargate (FastAPI Services)

Engineering Stack

  • FastAPI backend services
  • AWS ECS Fargate container deployment
  • Application Load Balancer with HTTPS
  • S3 object storage
  • PostgreSQL database (RDS / Supabase)
  • SQLAlchemy ORM
  • Stripe subscription system

Architecture Principles

  • Event-driven processing
  • Independent modular services
  • Idempotent data operations
  • Auditability and reporting
  • Schema validation enforcement
  • Production observability
